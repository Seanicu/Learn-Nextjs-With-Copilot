# Learn-Nextjs-With-Copilot 项目简介

## 项目概述

本项目用于学习 Nextjs 框架的使用、TS 和 TSX 语法、react 用法。适合有编程基础的人。

## 使用方法

1. 在本地使用 VS Code 打开本项目（也可以使用其他 IDE，但可能需要手动将 `/.github/copilot-instructions.md` 文件添加为自定义的说明文件，具体添加方法随 IDE 而不同）。
2. 使用 `/instructions/learning-start.md` 中提供的模板指令与AI（例如 Copilot Agent）对话来开启学习。你也可以自定义对话内容。
3. 使用 `/instructions/learning-review.md` 中提供的模板指令来让AI检查你的答案。你也可以自定义对话内容。

## 此外

- 你可以自行修改 `catalog.md` 或使用 `instructions/generate-catalog.md` 中提供的模板指令重新生成。
- 以本项目为模板，也可方便地修改为其他用途，例如学习外语
